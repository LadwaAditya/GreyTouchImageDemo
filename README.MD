Add this in your root build.gradle at the end of repositories:
```gradle
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
Step 2. Add the dependency in your module's gradle file
```gradle
dependencies {
	compile 'com.github.LadwaAditya:GreyTouchImageView:1.0'
}
```
